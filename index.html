📋 How to Use

For Certificate Issuers:

  1. Connect your MetaMask wallet
  2. Upload the certificate file (PDF/Image)
  3. Fill in recipient details
  4. Click "Issue Certificate" - this stores the hash on blockchain
  5. Share the QR code with the certificate recipient

For Verification:

  1. Upload the certificate file you want to verify
  2. Click "Verify Certificate"
  3. System checks if the file hash exists on blockchain
  4. Get instant verification results
